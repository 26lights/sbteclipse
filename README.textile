h1. sbt-eclipse

This project is a plugin for "SBT 0.9":http://github.com/harrah/xsbt/ providing a command to create "Eclipse":http://www.eclipse.org/ project files from a SBT project.

Attention: This is work in progress and so is SBT 0.9. In order to use this plugin you probably will have to build SBT 0.9 yourself.

h2. Usage

* Create a SBT project
* Add the plugin
* Now you can use the command _eclipse_ to create Eclipse project files
