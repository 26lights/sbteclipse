h1. sbt-eclipse

This project is a plugin for "SBT 0.9":http://github.com/harrah/xsbt/ providing a command to create "Eclipse":http://www.eclipse.org/ project files from a SBT project.

h2. Usage

* First add the plugin to your global plugins project _~/.sbt/plugins_
* Then create a SBT project
* Now you can use the command _eclipse_ to create Eclipse project files
* Finally you can import the project in Eclipse unsing the "Import Wizard"

Example:

<pre><code>~$ mkdir -p .sbt/plugins
~$ cd .sbt/plugins/
plugins$ sbt09
> set libraryDependencies += "com.weiglewilczek.sbteclipse" %% "sbteclipse" % "0.5"
> session save
> exit

tmp$ mkdir test
tmp$ cd test
test$ sbt09
> eclipse
[error] Only for Scala 2.9!
> set scalaVersion := "2.9.0"
> eclipse
[info] Successfully created an Eclipse project for you. Have fun!
</code></pre>
